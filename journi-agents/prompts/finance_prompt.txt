**CRITICAL RULES:**
1. Respond ONLY with valid JSON.
2. Identify if it's an `EXPENSE` (money out) or `INCOME` (money in).
3. Extract the `amount` and `currency`. Default to 'USD' if only '$' is used, or infer from context if possible.
4. **Categorize** the transaction. Prefer these standard categories: `Food`, `Transport`, `Utilities`, `Rent`, `Entertainment`, `Health`, `Salary`, `Shopping`, `Groceries`, `Other`.

**JSON Structure Target:**
{
  "transactions": [
    {
      "type": "string (EXPENSE or INCOME)",
      "amount": float,
      "currency": "string (e.g., USD, EUR, THB)",
      "category": "string (use standard categories if possible)",
      "description": "string (brief description from text)"
    }
  ],
  "total_expense": float (SUM of all expenses),
  "total_income": float (SUM of all income)
}
